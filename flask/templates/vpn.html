使用Docker + IPsec + Centos搭建VPN服务器


首先，需要选择一个服务器供应商，这里选择了Vultr，原因是操作简单，价格便宜，按时收费


新增服务器


选择亚洲地区的服务器（延迟低），操作系统选择CentOS 7 x64



选择好服务器规格后，点击Deploy Now，然后等待系统安装和部署，大概5-10分钟


在Products可以看到新增的服务器，点进去是服务器的详细信息



Mac OS可直接在终端使用SSH登录服务器



成功登入服务器后，下一步是安装docker

删除旧版本docker

sudo yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-engine


安装yum-utils，创建repository

sudo yum install -y yum-utils

sudo yum-config-manager \
    --add-repo \
    https://download.docker.com/linux/centos/docker-ce.repo


安装最新版本的Docker Engine，containerd和Docker Compose

sudo yum install docker-ce docker-ce-cli containerd.io docker-compose-plugin


启动docker

sudo systemctl start docker



使用以下命令快速搭建 IPsec VPN 服务器

docker run \
    --name ipsec-vpn-server \
    --restart=always \
    -v ikev2-vpn-data:/etc/ipsec.d \
    -v /lib/modules:/lib/modules:ro \
    -p 500:500/udp \
    -p 4500:4500/udp \
    -d --privileged \
    hwdsl2/ipsec-vpn-server

配置使用 IKEv2 VPN

查看容器的日志以获取 IKEv2 配置信息

docker logs ipsec-vpn-server


将配置文件复制到Docker主机

# 查看容器内的 /etc/ipsec.d 目录的文件
docker exec -it ipsec-vpn-server ls -l /etc/ipsec.d
# 示例：将一个客户端配置文件从容器复制到 Docker 主机当前目录
docker cp ipsec-vpn-server:/etc/ipsec.d/vpnclient.p12 ./



不同设备下配置 IKEv2 VPN 客户端


Windows 7, 8, 10 和 11

1.将生成的 .p12 文件安全地传送到你的计算机

2.右键单击 ikev2_config_import.cmd 并保存这个辅助脚本到与 .p12 文件相同的文件夹

3.右键单击保存的脚本，选择“属性”。单击对话框下方的“解除锁定”，然后单击“确定”

4.右键单击保存的脚本，选择“以管理员身份运行”并按提示操作

要连接到VPN：单击系统托盘中的无线/网络图标，选择新的VPN连接，然后单击“连接”


OS X (macOS)

首先，将生成的 .mobileconfig 文件安全地传送到你的Mac，然后双击并按提示操作，以导入为macOS配置描述文件。如果你的Mac运行macOS Big Sur或更新版本，打开系统偏好设置并转到描述文件部分以完成导入。在完成之后，检查并确保 "IKEv2 VPN" 显示在系统偏好设置 -> 描述文件中

1.打开系统偏好设置并转到网络部分

2.选择与你的VPN服务器IP（或者域名）对应的VPN连接。

3.选中“在菜单栏中显示VPN状态”复选框

4.单击“连接”


IOS

首先，将生成的 .mobileconfig 文件安全地传送到你的iOS设备，并且导入为iOS配置描述文件，在完成之后，检查并确保"IKEv2 VPN"显示在设置 -> 通用 -> VPN与设备管理（或者描述文件）中

1.进入设置 -> VPN。选择与你的VPN服务器IP（或者域名）对应的VPN连接

2.启用VPN连接


Android

将生成的 .sswan 文件安全地传送到你的Android设备

1.进入设置 -> VPN。选择与你的VPN服务器IP（或者域名）对应的VPN连接

2.下载并安装strongSwan VPN客户端

3.启动strongSwan VPN客户端

4.单击右上角的“更多选项”菜单，然后单击“导入VPN配置”

5.选择你从服务器传送过来的 .sswan 文件

6.在“导入VPN配置”屏幕上，单击“从VPN配置导入证书”，并按提示操作

7.在"选择证书"屏幕上，选择新的客户端证书并单击“选择”

8.单击“导入”

9.单击新的VPN配置文件以开始连接
